# Arduino_brazo_robotico_cabezaLaser

Brazo de tres servos con cabezal láser y buzzer, transmite texto a código morse mediante puntos láser y audio.
El brazo tiene 3 velocidades y función de retorno
Montado en un atmega328p con un circuito para el láser y otro para el brazo y el resto.
El láser lo saque desarmando un puntero láser, lo conecte a la salida de 3,3v la de 5V a lo mejor lo quema.
Para el encendido y apagado del láser hice un circuito con mosfet, diodo ss14 y resistencia de 10K, tiraba bien.
Todo con el IDE de Arduino.
